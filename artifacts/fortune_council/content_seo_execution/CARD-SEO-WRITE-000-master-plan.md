## 背景判斷

Click108 強項：

- 文章量大。
- 分類和熱門文內鏈密度高。
- RSS 可深分頁，發布節奏穩。
- 標題公式穩定，容易掃讀。

Click108 弱項：

- sitemap 不完整或不可用。
- 多數頁面 meta description 為空。
- 分類頁 canonical 缺。
- 文章頁缺 Article JSON-LD 與 FAQPage。
- H1 有雜訊。
- 內容偏運勢流量，不是搜尋答案結構。

Pantheon 超車方向：

- 先搶解釋型、問題型、FAQ 型長尾詞。
- 每篇都有 answer、FAQ、Article JSON-LD、FAQPage、Breadcrumb。
- 內鏈用產品線 + 五大情境 + 同分類 + 跨分類組成內容網。
- 不用保證式運勢承諾，不碰醫療、法律、投資建議。

## 必交付

產出 `artifacts/fortune_council/content_seo_execution/result.md`，包含：

1. 30 天撰文計劃。
2. 60 天 cluster 擴張計劃。
3. 90 天 Search Console / CTR / 內鏈優化計劃。
4. 第一批 30 篇的優先順序。
5. 每篇文章的固定規格：title、description、answer、FAQ、內鏈、CTA。
6. 不做事項。

## 驗收條件

- 計劃能直接派給撰文者，不需要再問「要寫什麼」。
- 每一批都有數量、類型、產出格式、驗收條件。
- 明確標示哪些是短期要贏 Click108 的槓桿。
- 不承諾排名保證。
