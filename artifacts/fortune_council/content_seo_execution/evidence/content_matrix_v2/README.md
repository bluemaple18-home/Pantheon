# Pantheon Content Matrix V2

這份矩陣提供每日單篇發文 Queue 的第二期題庫。生成來源是
`scripts/generate_content_matrix_v2.py`，不得手動修改 JSON；修改維度後應重新生成並執行
`--check`。

## 粒度

每一列就是一篇文章，而且只處理一個情境：

- 單體文：一個星座、MBTI、塔羅牌、紫微主星或八字十神 × 一個生活情境。
- 配對文：一組 MBTI 或星座配對 × 一個相處情境。
- 五個情境必須拆成五篇，禁止在一篇裡做五情境總整理。

## 規模

| 家族 | 計算方式 | 篇數 |
| --- | ---: | ---: |
| 星座單體 | 12 × 5 | 60 |
| MBTI 單體 | 16 × 5 | 80 |
| 塔羅單體 | 78 × 5 | 390 |
| 紫微主星單體 | 14 × 5 | 70 |
| 八字十神單體 | 10 × 5 | 50 |
| MBTI 配對 | 136 個無方向組合 × 5 | 680 |
| 星座配對 | 78 個無方向組合 × 5 | 390 |
| 合計 |  | 1,720 |

單體情境為感情、工作、人際、金錢、低潮。配對情境為戀愛、朋友、工作、衝突、
長期相處。

## 內容與去重契約

- ID、primary keyword 與 title 必須唯一。
- 配對用固定順序建立，避免 `A × B` 與 `B × A` 互相競食。
- 標題可以回答「適合嗎」，正文不得把類型或命理訊號直接判成適合、不適合或必然結果。
- 每日管線仍會用 registry 的 ID 與語意等價規則去重，已有文章不會重新排入。
- 每次仍只發布一篇，且必須通過 Gemini Writer、獨立 Reviewer 與 deterministic gates。
