# Pantheon SEO 撰文派工 Handoff

## Root Question

如何用可量產、可驗收、符合 Pantheon 邊界的文章規格，在短期內補足內容量與內鏈，並用更乾淨的 SEO / AEO / GEO 結構壓過 Click108？

## 目前狀態

- 已有競品研究工具：`scripts/competitor_seo_tool.py`
- 已有競品工具說明：`docs/competitor_seo_tool.md`
- 已有公開文章展出規範：`docs/pantheon_article_publication_standard.md`
- 已有 SEO keyword matrix：`artifacts/fortune_council/content_seo_matrix/keyword_seed_matrix.md`
- 已有第一批 30 篇 brief：`artifacts/fortune_council/content_seo_matrix/article_briefs_first_30.md`
- Click108 小樣本輸出：`output/competitor_seo/news.click108.com.tw/`

## Blocker

沒有外部搜尋量工具數據。短期先用既有 keyword matrix + 競品覆蓋缺口做內容生產；後續再用 Search Console / Ads Keyword Planner / Ahrefs / Semrush 驗證排序。

## Fork

本 handoff 只派「撰文計劃與內容產出規格」，不派前端開發、不派部署、不派 sitemap 實作。
