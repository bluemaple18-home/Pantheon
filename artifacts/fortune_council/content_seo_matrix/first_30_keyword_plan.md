# 第一批 30 篇關鍵字進攻規劃

任務ID：CARD-FORTUNE-FIRST-30-KEYWORD-PLAN-001
範圍：第一批文章的關鍵字規劃，不寫正文、不開發頁面

## 1. 規劃原則

每篇文章都要有：

```text
主攻關鍵字：標題和第一段一定要出現。
次要關鍵字：小標題、段落、常見問答要自然出現。
長尾問句：用來做常見問答和答案引擎段落。
收攏方向：導到免費入口或五大主題小報告。
```

限制：

```text
這是進攻規劃，不是搜尋量驗證。
真正排序優先度還要用關鍵字工具查月搜尋量、競爭度、相關問題。
```

## 7. 每篇文章關鍵字放置規則

```text
H1: 必放主攻關鍵字
第一段前 80 字: 必放主攻關鍵字一次
H2: 至少 2 個 H2 放次要關鍵字
FAQ: 放 3 到 5 個長尾問句
結尾 CTA: 放收攏入口與付費導向
內部連結錨文字: 優先用關鍵字，不用「點這裡」
```

## 8. 不能做的事

```text
不要每篇都塞同一組關鍵字。
不要用恐嚇式標題。
不要在財富文承諾投資或賺錢結果。
不要在感情文承諾復合。
不要把 MBTI 寫成心理診斷。
不要把塔羅寫成預言。
不要把命盤寫成絕對命運。
```
